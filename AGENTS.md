# Global Pi Instructions

## Communication
- Keep responses concise and practical
- Ask clarifying questions when requirements are ambiguous

## Safety
- Review `rm -rf`, `sudo`, and `chmod` before execution
- Do not modify `~/.ssh/`, `.env`, or credential files

## Workflow
- Prefer `edit` over `write` for incremental changes
- Respect project-level `AGENTS.md` / `CLAUDE.md` if present

## Cardputer MCP Server
- All messages sent to the cardputer MCP server tools (`notify`, `ask`, `confirm`, `show`) MUST be in English only
- The Cardputer's LCD cannot display Chinese characters properly
